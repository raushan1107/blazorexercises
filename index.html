Learning Path 1: Blazor Foundations

Lab 1 – Create Your First Blazor Application

Objective: Create and run your first Blazor Server application.

Step 1: Install .NET 6 SDK

Verify installation:


dotnet --version

If not installed, download .NET 6 SDK from official Microsoft website.

Step 2: Create Project


dotnet new blazorserver -n FirstBlazorApp
cd FirstBlazorApp
dotnet run

Step 3: Open in Browser

Navigate to:

https://localhost:5001

Expected Result: You should see the default Blazor template running.

What You Learned: How to create and run a Blazor Server app.

Lab 3 – Build Todo List

Objective: Build a basic Todo application.

Step 1: Create New Razor Page

Create Pages/Todo.razor

Step 2: Add Below Code


@page "/todo"

<input @bind="newTask" />
<button @onclick="AddTask">Add</button>

<ul>
@foreach (var item in todos)
{
    <li>@item</li>
}
</ul>

@code {
    string newTask;
    List<string> todos = new();

    void AddTask()
    {
        if (!string.IsNullOrWhiteSpace(newTask))
        {
            todos.Add(newTask);
            newTask = "";
        }
    }
}

Step 3: Add Navigation Link

Update NavMenu.razor to include Todo link.

Expected Result: You can add tasks dynamically.

Concept Covered: Loops, state management, UI re-rendering.

Lab 6 – Real-Time Sync with SignalR

Objective: Enable real-time updates across browsers.

Step 1: Create Service (Services/TodoService.cs)


public class TodoService
{
    public List<string> Todos { get; } = new();
    public event Action OnChange;

    public void Add(string task)
    {
        Todos.Add(task);
        OnChange?.Invoke();
    }
}

Step 2: Register in Program.cs


builder.Services.AddSingleton<TodoService>();

Step 3: Inject into Todo Page


@inject TodoService todoService

protected override void OnInitialized()
{
    todoService.OnChange += StateHasChanged;
}
Open two browsers. Add task in one window. Observe instant update in second window. Blazor Server uses SignalR internally for real-time communication.

Concept Covered: Dependency Injection, Events, Real-Time UI updates.
