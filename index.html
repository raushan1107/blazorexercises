Lab 3 – Build Todo List

Create a simple Todo list with add functionality.

@code {

Lab 6 – Real-Time Sync with SignalR

Now we enable real-time synchronization between multiple browsers using Blazor Server.

Step 1: Create Shared Service


public class TodoService
{
    public List<string> Todos { get; } = new();
    public event Action OnChange;

    public void Add(string task)
    {
        Todos.Add(task);
        OnChange?.Invoke();
    }
}

Step 2: Register in Program.cs


builder.Services.AddSingleton<TodoService>();

Step 3: Inject into Page


@inject TodoService todoService

protected override void OnInitialized()
{
    todoService.OnChange += StateHasChanged;
}
Open two browser windows. Add a task in one window. You will observe automatic update in the second window. This happens because Blazor Server uses SignalR internally to maintain a real-time connection.
