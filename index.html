Lab 6 – Real-Time Sync with SignalR (Corrected)

Objective: Enable real-time updates across browsers.

Step 1: Service


public class TodoService
{
    public List<string> Todos { get; } = new();
    public event Action? OnChange;

    public void Add(string task)
    {
        Todos.Add(task);
        NotifyStateChanged();
    }

    private void NotifyStateChanged()
    {
        OnChange?.Invoke();
    }
}

Step 2: Register in Program.cs


builder.Services.AddSingleton<TodoService>();

Step 3: Subscribe Safely in Component


@inject TodoService todoService
@implements IDisposable

protected override void OnInitialized()
{
    todoService.OnChange += HandleChange;
}

private async void HandleChange()
{
    await InvokeAsync(StateHasChanged);
}

public void Dispose()
{
    todoService.OnChange -= HandleChange;
}
⚠ In Blazor Server, UI updates must run on the correct Dispatcher thread. We use InvokeAsync(StateHasChanged) to safely refresh UI across browser circuits.
