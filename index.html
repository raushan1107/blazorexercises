Lab 3 â€“ Build Basic Todo Application

Objective: Understand loops and state.

Step 1: Create New Page

Create file: Pages/Todo.razor

Step 2: Add Below Code


@page "/todo"

<h3>Todo List</h3>

<input @bind="newTask" />
<button @onclick="AddTask">Add</button>

<ul>
@foreach (var item in todos)
{
    <li>@item</li>
}
</ul>

@code {
    string newTask;
    List<string> todos = new();

    void AddTask()
    {
        if (!string.IsNullOrWhiteSpace(newTask))
        {
            todos.Add(newTask);
            newTask = "";
        }
    }
}

Step 3: Add Navigation Link

Open Shared/NavMenu.razor

Add:


<NavLink href="todo">Todo</NavLink>

Step 4: Run & Test

  • Add a task
  • Observe it appears in list
ðŸ’¡ The foreach loop renders UI dynamically from C# list.
