Lab 1: Create Your First Blazor Application

Objective

In this lab, you will create your first Blazor Server application and understand the project structure.

Prerequisites

  • .NET 6 SDK installed
  • Visual Studio 2022 or VS Code

Step 1: Create Project


dotnet new blazorserver -n FirstBlazorApp
cd FirstBlazorApp
dotnet run

Lab 3: Real-Time Todo Sync using SignalR

Objective

Enable real-time synchronization across multiple browsers using Blazor Server.

Step 1: Create Shared Service


public class TodoService
{
    public List Todos { get; } = new();
    public event Action OnChange;

    public void Add(string title)
    {
        Todos.Add(new TodoItem { Title = title });
        OnChange?.Invoke();
    }
}

Step 2: Register Service


builder.Services.AddSingleton();

Step 3: Inject in Todo Page


@inject TodoService todoService
Open two browsers and add tasks. You will observe real-time updates powered by SignalR.
